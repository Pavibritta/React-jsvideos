# 🎥 Video Listing App (Mimic Server API)

This project demonstrates how to fetch and display video data from the **Mimic Server API** using React.  
It includes **search functionality** and **pagination**, implemented with best practices using `useEffect` and state management.

---

## 🚀 Features

- Fetch videos from Mimic Server API
- Search videos using keywords
- Pagination with Previous / Next controls
- Reset pagination on new search
- Loading & error handling
- Clean and responsive UI

---

## 🛠️ Tech Stack

- React (Hooks)
- JavaScript (ES6+)
- Fetch API
- Tailwind CSS
